The Plugin handles administration, display and printout for weekly lunches of a restaurant.

== Description ==

Tagesteller ist ein Plugin, welches Ihnen die Verwaltung Ihrer Mittagsmenüs pro Woche vereinfacht.
Das Plugin befindet sich in einer Basisversion und soll in Zukunft für Kundenwünsche angepasst werden.
Ich bin daher dankbar über jedes feedback: feedback@tagesteller.entsteht.at

Die Anzeige für das Frontend wurde für Bootstrap optimiert.

Zukünftige Features

erste Welle bis Jahresende 2015
*   Autovervollständigung beim Eintragen
*   Farbeinstellung für aktuellen Tag im Frontend
*   Restaurant-Logo für den Ausdruck hinzufügen
*   Verschiedene Formate für den Druck
*   verbesserung mobile Ansicht

weiters...
*   Bilder zu Speisen
*   Abonierfunktion für Gäste
*   Usability Verbesserungen auf Grund von Comunity-Feedback

Besonderer Hinweis:

Um den Service zu verbessern, werden Sie gebeten dem Plugin zu erlauben nach Hause zu telefonieren.
Ist die Erlaubnis erteilt wird Ihre Admin E-Mail Adresse, sowie die Websiteadresse bei uns hinterlegt werden.

== Installation ==

1. Laden Sie das Plugin herunter
2. Speichern Sie den Ordner Tagesteller im /wp-content/plugins Verzeichnis auf Ihrem Server
3. Gehen Sie ins WP Backend und dann auf Plugins
4. Aktivieren Sie das Plugin
5. Tragen Sie die Tagesteller der aktuellen Woche ein
6. Fügen Sie den Shortcode [tagesteller] auf der Seite ein, wo das Menü angezeigt werden soll.
